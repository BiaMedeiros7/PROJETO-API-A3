# Kanban API - CRUD de Tarefas

Este projeto contém uma API desenvolvida com FastAPI para gerenciar um quadro de tarefas baseado na metodologia Kanban. 

![imagem-trabalho.jpeg](https://github.com/BiaMedeiros7/PROJETO-API-A3/blob/main/imagem-trabalho.jpeg)

>>Tecnologias:
 Python
 FastAPI
 SQL Server

>>Bibliotecas:
SQLAlchemy 
Pydantic 
Uvicorn 
Pytest 


# Como executar: 
1. Clone o repositório
2. Crie um ambiente virtual e ative-o
3. Instale as dependências
4. Configure o banco de dados em /banco-de-dados-api_tarefas_api.sql
5. Inicie a API
6. Acesse a documentação interativa da API em
   http://127.0.0.1:8000/docs#/


Sinta-se à vontade para abrir issues e pull requests para melhorias no projeto!
